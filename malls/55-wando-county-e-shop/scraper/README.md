# 완도군이숍 Scraper (ID: 55)

## Overview
Web scraper for 완도군이숍 (https://wandofood.go.kr/)

## Requirements
- Node.js 16+
- npm or yarn

## Installation
```bash
npm install
```

## Usage
```bash
npm run scrape
```

## Configuration
See `config.json` for scraper settings.

## Output
Scraped data will be saved to the `data/` directory with timestamp.
